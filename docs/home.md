# Iris Static Site Generator

Iris is a lightweight static site generator designed for building clean, fast documentation — no bloat, no complex configuration, just structured content that works.

![iris_ui](./assets/icons/iris_ui.png)

Whether you're documenting an internal tool, an open-source library, or a personal project, Iris gives you a focused writing environment and a polished result out of the box.

### Get started in minutes
Clone the repository, drop your Markdown files into docs/, and you're done. No build pipeline to configure, no dependencies to wrestle with.
```bash
git clone https://github.com/stepan-vana/iris-ssg.git
```

After that:

```bash
cd iris-ssg
```

### Built to be yours
Iris doesn't lock you in. The structure is intentionally minimal — tweak the styles, swap the icons, replace the branding. Everything is where you'd expect it to be.

### Open source
Iris is publicly available on GitHub. Found a bug? Have an idea? Open an issue or submit a pull request — contributions are welcome.
