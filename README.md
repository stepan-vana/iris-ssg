## Documentation, without the overhead.

Iris is a lightweight static site generator for clean, fast docs —
no build pipeline, no complex config, no lock-in.

---

### Up and running in minutes.

Clone the repo, drop your Markdown into `docs/`, open the browser. Done.

```bash
git clone https://github.com/stepan-vana/iris-ssg.git
cd iris-ssg
```

1. **Clone the repo** — everything you need is already there.
2. **Add your content** — drop Markdown files into `docs/`.
3. **Run Live Server** — start a local development server to preview your site instantly.

---

### Built to be yours.

Iris doesn't lock you in. Minimal by design — tweak the styles,
swap the icons, replace the branding. Everything is where you'd expect it.
